Installation

Follow the steps below to get started with your Site Template:

  1. Open the Kanye Folder to find all the Template Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • index.html - Index File/Homepage
    • assets/scss - SCSS Folder
    • assets/css - CSS Folder
    • assets/img - Images Folder
    • assets/js - Javacripts Folder
    • assets/vendor - vendor Folder
  4. You're now good to go..! Start adding your Content and show off your Beautiful Personal Website in style.

Customization

Changing Color Skin

You can change easily color with sass variable.

You can set your primary and secondary color in variable.scss file(\assets\style\scss) 

                $px-theme:       #1877F2 !default; // Main theme color

you can use Koala software for scss compile

Or You can compile via gulp

Setting up Dev Tools

Unzip the theme package to any folder and open a command line at that location.

Kanye's dev tools require NodeJs and the Gulp CLI package. Assuming you have Node and Git installed, the following command installs Gulp Command Line Utility globally so the gulp command can be used anywhere in your system.

npm install gulp-cli -g

When you have installed Gulp, run npm install and the required packages for the Kanye dev tools will be downloaded to the node_modules directory.

npm install

If you see errors such as EACCESS during installation, please follow these instructions at npm docs to install Node via nvm which resolves the permissions errors.

After installing npm you can run gulp sass command. This command help you to compail SCSS to CSS

gulp sass

Now you are ready to go
